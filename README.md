# what is troll text ?

It is a command-line tool 
to facilitate the creation or replication 
of the write mode of Homestucks trolls

# how use ?
arg 1 is for the key replacement file
arg 2 is for the text

```console
./troll_text './example.txt' 'hello world'
```

the result will be

```console
|-|3ll0 w0rld
```

# how build ?

just install the cmake , gcc and g++
and run:

```console
cmake .. ; make
```
